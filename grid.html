A Proper Bootstrap Grid Example

Resize your browser window to see the grid in action!

1. Equal-width Columns

Using the basic .col class, Bootstrap will automatically create equal-width columns. Add more .col divs and they will adjust accordingly.

2. Responsive Columns (The most important concept!)

Use classes like .col-md-6 to control layout. These columns will be side-by-side on medium (md) screens and up, but will stack vertically on small screens (like a mobile phone).

.col-md-6
On mobile, I take up the full width.

On mobile, I stack below the first column.

3. Setting Specific Widths

You can define a specific size for a column out of the 12 available units per row. Here we have a .col-6 (half-width) and two .col-3 (quarter-width) columns.

.col-6 (50% width)
.col-3 (25% width)
.col-3 (25% width)

4. Offsetting Columns

Use .offset-md-* classes to push columns to the right, creating empty space for alignment.

.col-md-4
.col-md-4 .offset-md-4
.col-md-3 .offset-md-3
.col-md-3 .offset-md-3
